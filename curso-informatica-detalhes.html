Curso de Informática Básica

Sobre o Curso

Aprenda os fundamentos essenciais da informática para se destacar no mercado de trabalho. Este curso foi desenhado para iniciantes e aborda desde o uso do sistema operacional até a criação de documentos, planilhas e apresentações profissionais.

O que você vai aprender:

  • Navegar com segurança e eficiência na internet.
  • Principais Teclas do teclado.
  • Utilizar editores de texto (como Microsoft Word) para criar documentos.
  • Criar e gerenciar planilhas eletrônicas (como Microsoft Excel).
  • Desenvolver apresentações de slides impactantes (como Microsoft PowerPoint).
  • Gerenciar arquivos e pastas no computador.
